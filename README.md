# SharePoint Online & OneDrive PowerShell Scripts

A collection of practical PowerShell scripts for SharePoint Online and OneDrive administration, troubleshooting, and automation.

## Requirements
- PowerShell 7+
- PnP.PowerShell module
- SharePoint Online / OneDrive admin permissions

## Repository Structure
- SharePointOnline/ – SPO-focused scripts
- OneDrive/ – OneDrive for Business scripts
- Utilities/ – Shared helper functions

## Usage
Each script contains built-in help:
```powershell
Get-Help .\ScriptName.ps1 -Detailed
``
